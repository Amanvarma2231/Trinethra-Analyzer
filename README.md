# 🛡️ Trinethra: Supervisor Feedback Analyzer
**100% Professional-Ready AI Behavioral Intelligence Suite**

Trinethra is a high-performance analysis tool designed to extract deep behavioral insights from supervisor-fellow feedback transcripts. It uses advanced LLMs to score performances, map business KPIs, and identify training gaps with surgical precision.

---

## ⚡ Quick Start (One-Click Launch)
To run the entire ecosystem (Backend + Frontend + Browser) automatically:
```bash
python run.py
```
*This master script cleans up ports, installs missing dependencies, and orchestrates both servers.*

## 🚀 Key Features
- **🌓 Dual Theme Mode:** Toggle between "Sleek Dark" and "Professional Light" themes.
- **📈 Fellow Trend Analysis:** Automatically tracks progress across multiple sessions.
- **🎤 Voice-to-Text Input:** Dictate feedback directly using AI speech recognition.
- **🧠 Multi-Step Reasoning:** Sequential AI analysis for Rubric Scoring, Evidence, and KPI Mapping.
- **📄 Professional PDF/CSV Export:** Generate branded reports and raw data sheets.
- **🔎 History Search & Highlighting:** Instantly find past sessions and jump to specific quotes.

## 🛠️ Tech Stack
- **Backend:** FastAPI (Python 3.12), Pydantic V2, SQLite.
- **Frontend:** Vanilla JS, CSS3 (Advanced Glassmorphism), HTML5.
- **AI Engine:** Ollama (Llama 3.2 / Mistral) with Mock Mode for demo.
